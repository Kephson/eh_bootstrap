Administrator Manual
====================

Target group: **Administrators**

Download and install TYPO3 CMS. Install the TYPO3 bootstrap_package extension to test the extension.


.. _admin-installation:

Installation
------------

- Just install the extension via extension manager or via composer
- All files will be included automatically in a correct way if you're using the bootstrap_package

To install the extension, perform the following steps:

#. Go to the Extension Manager
#. Install the extension
#. Load the static template
#. ...
